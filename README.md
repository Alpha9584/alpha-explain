## Installation

### 1. Get a Google Gemini API Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Create a new API key (it's free!)
4. Copy the API key

### 2. Load the Extension in Chrome

1. Open Chrome and navigate to `extensions`
2. Enable **Developer mode** (toggle in the top right)
3. Click **Load unpacked**
4. Select the `Œ±-explain` directory
5. The extension should now appear in your extensions list

### 3. Configure the API Key

1. Click the Œ±-explain extension icon in your toolbar
2. Paste your Gemini API key
3. Click **Save Settings**

## Usage

1. Navigate to any webpage
2. Highlight a scientific term (e.g., "photosynthesis", "quantum entanglement", "mitochondria")
3. Press `Shift + E`
4. Read the explanation in the beautiful popup!
5. Press `Esc` or click outside to close
